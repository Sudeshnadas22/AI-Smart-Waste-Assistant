# ♻️ AI Smart Waste Assistant

An AI-powered waste classification system built using MobileNetV2 and Streamlit.

## Features

* Classifies waste into:

  * Cardboard
  * Glass
  * Metal
  * Paper
  * Plastic
  * Trash
* Displays confidence score
* Identifies recyclable and non-recyclable waste
* Provides disposal instructions
* Web application built with Streamlit

## Model Performance

* Test Accuracy: 81.7%
* Transfer Learning: MobileNetV2
* Dataset: TrashNet (2527 images)

## Tech Stack

* Python
* TensorFlow / Keras
* MobileNetV2
* Streamlit
* NumPy
* Matplotlib
